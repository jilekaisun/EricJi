  2. 电源与电量状态

    • 相机使用12V 5A 的DC 接口电源适配器
    • 相机使用5000mAh 的可更换电池，不开 WiFi 的情况下续航约75min
    • 关机充电时
      • 当相机未充满电时，电源指示灯红灯长亮
      • 当相机充满电时，电源指示灯绿灯长亮
    • 开机充电时
      • 当相机电量低于10%时，电源指示灯显示为红灯长亮，直至低点保护或完全无电自动关机
      • 当相机电量处于11%~20%时，电源指示灯显示为黄灯亮
      • 当相机电量处于21%~100%时，电源指示灯显示为白灯亮
      • 当相机进行具体工作时，工作灯颜色将会和电源指示灯的颜色保持一致，具体工作状态灯的闪烁状态见相机操作使用一节

  3. 存储

    • 相机支持视频速度等级在 v30及以上更高速度规格的SD卡。
      推荐闪迪SD卡（京东购买链接：https://item.jd.com/530218.html#crumb-wrap 与 亚马逊的购买链接:https://www.amazon.com/dp/B01K3X3G4U/ref=cm_sw_r_cp_dp_T1_x-HnzbBNH82XP ），卡格式推荐 exFat，对存储卡容量没有限制。视频速度等级为 SD 卡协会推出的最新标准，速度标准如图：

    • 相机还支持使用 USB 3.0或者更高速的 SSD 移动硬盘。文件格式推荐 exFat，对容量没有限制。推荐使用三星 SSD T3固态硬盘（京东购买链接：https://item.jd.com/2365493.html 与亚马逊的购买链接：https://www.amazon.com/dp/B01AVF6WN2/ref=cm_sw_r_cp_dp_T1_DR1pzbNNT7SMX）

    • 当同时使用移动硬盘与 SD 卡时，拍摄的内容会默认优先存储至移动硬盘

    • 用户新插入一个存储设备时，在第一次发起录像时，相机将会提示进行速度检测。检测速度达标的情况下才可正常进行录像。用户也可以至相机设置的 Storage 功能中，手动对某个存储设备进行速度检测

  4. 使用 Insta360 Pro

    • 开机固件升级

      • 请务必确保相机电池电量在30%以上，并且接好电源适配器。
      • 然后将拷贝有最新固件的 SD 卡或移动硬盘插入相机。
      • 准备好以上步骤后，重启相机，开机过程中相机进入 Updating 状态，即正在更新固件。更新完毕后相机会自动重启。

      • 可以在相机的设置功能中，点击 Info 查看相机的系统版本以及序列号等信息

    • 关机

      • 相机处于拍摄模式的准备就绪、设置、异常提示等状态下，长按 Power 键3秒会执行关机
      • 相机处于拍摄中（拍照中、录像中、直播中）、存储中状态时，长按 Power 键盘3秒会先停止当前的工作状态并完成存储后再执行关机

    • 拍照功能

      1. 拍照模式界面

      2. 操作使用

        • 从首页选择拍照模式，Power 键确定进入，等待相机完成准备工作后，相机显示拍照 Ready 状态

        • 拍照模式处于准备就绪状态下，按上下切换键可以快速切换拍照的规格档位

        • 拍照模式处于准备就绪状态下，按一下 Power 键，即可按当前规格拍摄一张照片。拍摄完后如果是需要实时拼接的档位，则会进入 processing 状态，处理完毕后会完成存储，再变为拍照准备就绪状态。

每次拍照时，默认需要等待5s的倒计时，倒计时过程中工作灯将持续闪烁，且伴有音效提示，请在这5s内远离相机，尽量避免太过靠近相机，影响拍照效果。拍照成功时将有完成音效的提示，

       - 拍照模式下完成一张照片拍摄后如果检测到剩余空间不足100MB 时，则会提示剩余空间不足，无法再继续拍摄下一张照片。请及时更换有充足存储容量的存储卡
       ![video_alert_insufficient](media/14966545705192/video_alert_insufficient.jpg)

       - 拍照过程中如果相机的电量过低，将会在完成此次拍摄后提示电池电量过低，请立即接上电源适配器或者更换电量充沛的电池，否则无法再继续执行下一次拍摄
           ![all-alert-lowbattery](media/14966545705192/all-alert-lowbattery.jpg)

       - 拍摄过程中如果遇到了 Error 提示，则意味着操作未正常完成，请查阅相关错误说明，获得解决方法

   3. 拍照规格档位
       - 8K/OF —— 拍摄存储8K全景照片原片和光流实时拼接的照片
       - 8K/3D/OF —— 拍摄存储8K3D全景照片原片和光流实时拼接的照片
       - 8K —— 拍摄存储全景照片原片，后期合成8K全景/3D全景的照片
       - HDR —— 拍摄存储 HDR 的照片原片，可后期合成 HDR 的8K全景/3D全景的照片
       - RAW —— 拍摄存储 RAW 格式的照片原片，可后期合成8K全景/3D全景的照片
       - Burst —— 一键拍摄存储10组照片原片，可后期合成8K全景/3D全景的照片
       - Customize —— 用户可快速选择此项应用最近一次使用 APP 操控或者二维码扫描时保存的参数。使用详情见二维码扫描一节。

- 录像功能
   1. 录像模式界面
 ![录像界面](media/14966545705192/%E5%BD%95%E5%83%8F%E7%95%8C%E9%9D%A2.png)

   2. 操作使用
       - 从首页选择录像模式，Power 键确定进入，等待相机完成准备工作后，相机显示录像 Ready 状态
       - 录像模式处于准备就绪状态下，按上下切换键可以快速切换录像的规格档位
       - 录像模式处于准备就绪状态下，按下 Power 键，即可按当前规格开始一次录像。
       - 注意：如果使用的存储设备是第一次使用，则需要执行速度检测，确保卡速度达标之后才可正常录像。
       ![all-alert-speedtest-sdcard](media/14966545705192/all-alert-speedtest-sdcard.jpg) ![all-alert-speedtest-sdcarding](media/14966545705192/all-alert-speedtest-sdcarding.jpg) ![all-alert-speedtest-ok](media/14966545705192/all-alert-speedtest-ok.jpg)

       - 录像过程中再次按下 Power 键则会停止录像。如果是需要实时拼接的档位则在停止后会进入 Processing状态，处理完毕后会完成存储，再变为录像准备就绪状态。录像中工作状态指示灯将持续闪烁，直到拍摄并存储完成。如果会影响拍摄环境，可以在相机设置中关闭 LED 灯。录像结束存储完成时相机将有完成音效提示。
       - 录像中如果剩余空间不足1G 的情况下，会自动停止录像并保存当前的录制内容。如果再次尝试发起一次录像将会看到存储空间不足的提示，请及时更换有充足存储容量的存储卡。
       - 录像过程中如果相机的电量过低，将会在完成此次拍摄后提示电池电量过低，请立即接上电源适配器或者更换电量充沛的电池，否则无法再继续执行下一次拍摄
       - 录像过程中如果遇到了 Error 提示，则意味着操作未正常完成，请查阅相关错误说明，获得解决方法

    3. 录像规格档位
      - 8K/30F —— 8KPano@30fps，拍摄存储视频原片
        - 6K/3D/30F —— 6K3D@30fps，拍摄存储视频原片
        - 4K/120F —— 4K@120fps，拍摄存储高帧率原片
        - 4K/3D/60F —— 4K3D@60fps，拍摄存储高帧率原片
        - 4K/30F —— 4KPano@30fps，拍摄存储原片和实时拼接的视频
        - 4K/3D/24F —— 4K3D@24fps，拍摄存储原片和实时拼接的视频
        - 另外，在相机设置中可开启/关闭Flat Color选项，决定是否拍摄细节更广的视频内容
        - Customize —— 用户可快速选择此项应用最近一次使用 APP 操控或者二维码扫描时保存的参数。使用详情见二维码扫描一节。

- 直播推流功能
   1. 直播推流模式界面


   2. 操作使用
       - 从首页选择直播模式，Power 键确定进入，等待相机完成准备工作后，相机显示直播 Ready 状态
       - 直播模式处于准备就绪状态下，按上下切换键可以快速切换直播的规格档位
       - 直播模式处于准备就绪状态下，按下Power 键，会根据当前选择的档位进行推流直播或者 HDMI 输出直播。再次按下 Power 键会停止直播。直播工作状态指示灯将持续闪烁，直到拍摄并存储完成。如果会影响拍摄环境，可以在相机设置中关闭 LED 灯。

       ![live-state-recording](media/14966545705192/live-state-recording.jpg)
       - 直播过程中如果遇到网络异常使得推流异常中断，在不手动停止直播的情况下，相机会不断地尝试重连，自动恢复推流直播


       - 通过客户端可以设置推流地址、设置直播的同时将视频原片存储到本地。假如设置了直播的同时存储到本地，则会遵循录像模式相关会遇到的异常
       - 直播过程中如果相机的电量过低，将会自动停止直播并提示电池电量过低，请立即接上电源适配器或者更换电量充沛的电池，否则无法再发起下一次直播
       - 直播过程中如果遇到了 Error 提示，则意味着操作未正常完成，请查阅相关错误说明，获得解决方法

   3. 直播推流规格档位
      - 4K/30F —— 4KPano@30fps，网络直播推流。如果不设置推流地址，默认机内推流。可通过客户端软件设置边直播边存储原片到存储卡。
      - 4K/3D30F —— 4K3D@30fps，网络直播推流。如果不设置推流地址，默认机内推流。可使用客户端设置边直播边存储原片到存储卡，但最高帧率会降到24fps，同录像实时拼接。
      - 4K/30F HDMI —— 4KPano@30fps，HDMI 输出信号。可通过客户端设置边输出直播边存储原片到存储卡
      - 4K/3D/30F HDMI —— 4K3D@30fps，HDMI 输出信号。可使用客户端设置边直播边存储原片到存储卡，但最高帧率会降到24fps，同录像实时拼接。
      - Aerial —— 航拍模式，该模式下输出1920x960的 HDMI 全景画面给无人机进行预览，同时可存储每个镜头2560x1440分辨率的视频原片
      - Customize —— 用户可快速选择此项应用最近一次使用 APP 操控或者二维码扫描时保存的参数

  1. WiFi 开关

    • 具体查看相机设置一节的设置相机 wiFi 模式部分内容。

  2. 拼接校准

    • 相机的机内拼接效果取决于具体拍摄场景。比如相机在不同的远景和近景之间的效果会有差别。所以用户预览或者试拍一些作品发现实时拼接的效果（直播、录像实时拼接时）不满意，可以用相机的这个功能校准。进入该功能后请按提示在5秒内远离相机1米远，以便于倒计时结束后的拼接校准可以获得最好的效果。

  3. 相机设置

    1. 设置相机的 WiFi 模式

      • Pro 相机提供了两种 WiFi 连接的方式：自身 AP 热点和连接外部无线 WiFi

      • 设置为 AP，然后开启首页的 WiFi 按钮，则屏幕顶部会出现WiFi 开启的标志并且显示一个 IP 地址。此时用户使用客户端设备搜寻附近的相机 AP，默认密码88888888（8个8），可以进行连接。

      • 设置为 WiFi，即认为连接外部 WiFi 的模式，选中首页的 wifi 按钮，按下设置键，将进入等待用户扫描相机 WiFi 信息（SSID 与密码）的界面。用户可以使用手机 APP 的二维码生成功能，设置好要连接的外部WiFi 的 SSID 与密码，生成二维码给相机扫描。扫描成功后相机将自动连接至该外部网络。详情请见手机 APP 二维码扫描部分。

    2. 设置相机Ethernet 网线连接

      • DHCP：用于将 Pro 连入某个局域网络，动态获取网络内的 IP 地址
      • Direct：用于将 Pro和 PC/笔记本电脑 直接连接
        具体用法：
        1. 使用网线将 pro 与电脑的网口直接相连。如果需要连接的电脑没有网线接口，可以使用 pro 配备的网线与网卡接入电脑的 USB 接口。
        2. 将 pro 的相机设置中，Ethernet 选项切换为“Direct”模式。此时，相机顶部的 IP 地址将会固定显示为“192，168.1.188”
        3. 将另一端连接的电脑，有线网络的IP 地址获取方式设置为手动，将电脑与 Pro 设置在同一个网段。
          • Mac OS：
            ![](media/14966545705192/Mac Direct.png)
            • 打开网络偏好设置
            • 选择USB 网卡网络，将配置 IPV4选项设置为“手动”，将 IP 地址设置为“192.168.1.100”，子网掩码设置为“255.255.255.0”
            • 应用保存
          • Windows：
            ![](media/14966545705192/Win Direct.png)
            • 打开本地网络，右键菜单选择属性，在弹出界面中选择 IPV4地址设置。
            • 如图，将 IP 地址设置为“192.168.1.100”，子网掩码设置为“255.255.255.0”
            • 点击保存
        4. 完成以上步骤之后，可以照常的使用电脑端的 Pro 控制软件去操控相机。

    3. FlatColor：视频的 FlatColor 模式，开启此项后将会得到更适合后期处理的视频画面

    4. Frequency：用于设置视频画面的刷新频率。50Hz、60Hz，分别适合不同电力频率的地区使用

    5. Gyro Calibration：陀螺仪校正功能。Pro 相机支持陀螺仪数据记录用于拍摄、后期处理时的防抖与水平校正。如果用户预览、试拍时发现相机画面的陀螺仪水平防抖效果不好，可通过此功能来校正

    6. Storage: 存储管理，用户可使用此功能查看当前存储卡或者USB 存储设备的容量使用情况。还可以选择插入的设备进行格式化、速度检测等操作

    7. LED: LED 开关，用户可开启或关闭相机周身指示灯

    8. Speaker: Speaker 开关，用户可开启或关闭相机的扬声器提示音

    9. Bottom Logo：设置画面底部是否 Logo 遮挡

    10. Info: 查看相机硬件固件、系统等程序版本信息

    11. Reset: 将相机记录的Customize参数重置为出厂默认

  4. 如何设置与修改底部Logo？

    • 请在相机设置中将 Bottom Logo 选项开启。
    • 将命名为”_logo.png”的 Logo 图拷贝至相机当前使用的 SD 卡或移动硬盘。
    • 打开相机观看预览流或者拍摄一个实时拼接的内容，相机会自动将”_logo.png”转化成底部的圆形遮挡水印。
    • “_logo.png”要求宽高比最大不超过50%。在宽度固定的情况下，增大高度将增大底部 logo 的半径。

  5. HDMI 输出

    • 使用相机进入任意拍摄模式时，Ready 状态下通过 HDMI 接入显示设备将显示实时拼好的预览画面
    • 除HDMI 直播输出与拍摄模式Ready 的预览状态，其余状态下是无法输出 HDMI 预览画面

  6. 二维码扫描功能

    • 相机可以在不与手机APP连接的情况下，扫描识别 APP 生成的二维码，对准相机屏幕两侧的镜头的其中一个，等待相机识别后，即可将参数保存到 Customize 参数档位中，以便进行更多操作。相机扫码识别成功时将会播放识别成功的音效。

    • 在拍照、录像、直播推流的准备模式下，按下设置键，将会进入等待扫描二维码状态。请使用 APP 的二维码生成器，设置对应模式的参数，并生成二维码给相机识别。

    • 在首页导航栏中选中 WiFi 图标，按下设置键即可进入设置等待扫描外部 WiFi 信息的二维码状态。请使用 APP 的二维码生成器，设置外部 WiFi的 SSID 和密码，并生成二维码给相机识别。相机在识别成功后将自动连接填写的外部 WiFi 信息

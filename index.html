如何使用 GPS 扩展配件与上传内容至谷歌街景？

发表于 2017-11-09

1、请至官网商城 。
Pro 产品的套餐中提供了谷歌街景套餐，配套一个 GPS 信息模块，通过 Pro 底部的 USB 口扩展使用。

2、请确保相机固件版本已更新至 XXX 及以上。

3、请将 GPS 信息模块插入Pro 底部的 USB 口，在 Pro 正常运行下，确认 GPS 配件上的工作灯亮起，此状态下拍摄的内容将会记录 GPS 信息。

请注意，GPS 配件记录的信息在受到物体遮挡的情况下会变得不准确，建议使用USB延长线（套餐附赠）

4、如果是为了拍摄内容上传至谷歌街景，还需如下准备工作：

  • 拍摄的内容务必需要带有 GPS 信息。
  • 全景照片规格需要3840x1920及以上分辨率，画面比例为2:1
  • 全景视频规格需要3840x1920及以上分辨率，画面比例为2:1，且拍摄的视频不允许分段（拍摄前至相机设置将 Video Seg 置为 Off），不允许经过 Stitcher 的陀螺仪校正处理。推荐使用 Pro 的8K@5fps 这一分辨率制作视频，上传之后谷歌会将视频按图片序列的方式处理成一张张街景。
  • 在 Stitcher 顶部的文件（File）入口中选择上传内容至谷歌街景，登录您的谷歌街景账号，将符合以上要求的内容上传。上传成功之后，需要等待谷歌服务器处理完成才能显示在谷歌街景应用中。

5、向谷歌街景上传内容前，请先阅读并承诺遵守谷歌街景内容上传政策（英文）

如何使用 Insta360 Pro 进行微博全景直播

发表于 2017-10-08

一、首先访问微博直播创建工具（Pro专用）的网址：http://tools.insta360.com/tools/weibo
用您的微博账号获取授权，然后创建一个微博的全景直播。

注意：必须通过实名认证的微博账户才可创建全景直播。部分蓝V（企业用户）和海外 IP 的用户由于微博政策限制，无法创建全景直播，请与微博协商解决。

二、微博账号授权登录完成后即可点击创建直播。

  1. 创建一个全景直播需要填写“标题”、“微博内容”。
  2. 根据 Pro 客户端设置的分辨率选择创建工具上的分辨率选项，请务必保持一致，否则将无法正常创建直播。
  3. 用户可自定义封面图地址，要求图像比例与直播比例一直。如果不自定义将默认设置为官方的封面图
  4. 直播的公开性，可选择仅自己可见或者公开。需要注意的是，微博的内容设置为非公开的观看权限后，将无法再设置为公开。
  5. 设置完以上必要参数后，即可创建直播

三、创建完成的直播，信息将显示在工具中。用户可复制推流服务器地址（Server）与推流密钥（Key） 粘贴到客户端中。也可以结束该直播（直播状态将会从直播中变为直播结束的回放）或者删除该直播（直播状态将提示不存在，此操作无法撤销）

四、创建完成直播后，在 Pro 客户端在直播推流的模式下，选择自定义服务器推流，填写下方的推流服务器地址（Server）与推流密钥（Key），开启直播后将刚创建的微博直播进行推流。

五、经实测，微博目前的直播没有转码压缩，推多大规格的分辨率和码率就显示多大。为用户端流畅观看考虑，推荐使用3840*1920（4K）@4M 码率、2880x1440（3K）@2.5M 码率、1920x960（2K）@1.5M 码率的规格进行直播推流。

使用 APP 操控相机（以iOS版本为例）

发表于 2017-06-13

一、APP 兼容性要求：

  1. Pro iOS APP 适配机型要求：

    • 处理器为 A7及以上的 iOS 设备，如iPhone 5S及以上的 iPhone、iPad Air/Mini2及以上的 iPad、iPod Touch 6。不满足该要求的机型可能会存在预览画面无法正常播放、启动缓慢等问题，但基础的连接操控功能不受影响，请酌情使用。
    • iOS 系统版本要求8.0及以上。

  2. Pro Android APP 适配机型要求：

    • 要求Android 5.0及以上系统，建议使用2016年之后推出的主流旗舰机型。具体的兼容性列表后续推出。

二、与相机建立连接

  1. 手机 APP 提供了两种连接方式：

    • 局域网连接：
      将手机 APP 和相机接入同一个局域网，在 APP 上选择 Connection Method A，填写相机屏幕上的 IP 地址，然后点击连接，连接成功后将进入功能菜单

    • 相机 AP 连接：
      先将相机开启 AP 模式，再用手机扫描并连接对应的热点，一般为 Insta360-Pro-xxxxxx（x 部分为序列号后六位）的格式，默认 AP 连接密码是88888888（数字八个8）。在 APP 上选择 Connection Method B，然后点击连接，将默认与AP固定的“192.168.43.1”IP 建立连接，连接成功后将进入功能菜单

三、 功能操作

  1. 相机提供了如下功能入口：

    • 拍照
    • 录像
    • 直播推流
    • 拼接校准
    • 存储管理

    • 相机设置（待开发）

  2. 预览相机画面：

    • 点击进入拍照、录像、直播模式，APP 会自动加载预览流，待完成后可进行对应的拍摄：
    • 开启陀螺仪防抖，该按钮开启状态下，相机的实时拼接功能存储的视频画面，将自动根据陀螺仪数据进行纠正。注意，此功能在拍摄3D 的实时拼接内容或3D 直播时无法生效。
    • VR 模式预览按钮，点击切换至 VR 眼镜模式观看预览效果
    • 关闭预览画面按钮（关闭后可以省电，轻触预览区域再次开启）

  3. 拍照

    1. General（拍照基础设置）：

      • Mode（拍摄模式）：
        • Normal（普通拍摄）
        • RAW
        • HDR
        • Burst（一键快速十连拍）
      • Content Type（内容类型）：
        • 360° Pano
        • 360° 3D
      • Optical Flow（光流实时拼接选项）
      • Delay Timer（延迟拍摄定时器）

    2. Exposure（曝光设置）：

      • Mode：
        • Auto（自动曝光）
        • Manual（手动曝光）
        • Isolated Exposure（各镜头独立曝光）
      • 在自动曝光模式下可调整 EV、白平衡
      • 在手动曝光模式下可调整 ISO、Shutter
      • 在各镜头独立曝光模式下可调整 EV、白平衡
      • 可重置曝光参数为默认设置

    3. Property（画面属性设置）

      • Brightness（亮度）
      • Saturation（饱和度）
      • Sharpness（锐度）
      • Constrast（对比度）
      • 可重置画面属性参数为默认设置

    4. 参数值范围：

      • EV 范围：普通拍摄情况下是 -5~5，HDR 的可调整的 EV 差值范围是-3.9~3.9
      • ISO 范围：100~6400
      • Shutter范围：普通 Shutter 为1/2s~1/8000s，Long Shutter 状态下是1s~60s
      • WB 档位：Auto（自动白平衡）、2700K、4000K、5000K、6500K、7500K
      • Brightness（亮度）：-255~255
      • Saturation（饱和度）：0~255
      • Sharpness（锐度）：3~6
      • Constrast（对比度）：0~255

    5. 拍照功能归纳：

      • 拍摄360°全景/3D照片，设置Mode 为 Normal、选择 Content Type，设置 Optical Flow 光流实时拼接。相机将会在存储实时拼接的照片的同时，将6个镜头的原片分别也存储下来。如果不开启实时拼接，则只会存储6个镜头的原片。无论何种拍摄设置，相机都会存储4000x3000/镜头的原片，可在 Stitcher 中合成最高8K分辨率的全景/3D照片。
      • 拍摄 RAW、HDR、Burst时，没有光流实时拼接选项，只可以存储6镜头原片，用户可配合 Stitcher 进行处理。 RAW 为 DNG 格式，拍摄 HDR 时需要额外设置一个 HDR EV 差值，来控制过曝和欠曝的两组照片的 EV 差值。
      • 曝光参数的设置 中，Auto/Manual 是全局统一的自动/手动曝光。如果选择 Isolated Exposure，则6个镜头将会各自独立的自动曝光。独立曝光适合拍摄明暗差距较大的场景，比如拍摄艳阳高照的窗边与室内，比如舞台绚丽明亮但观众席较暗的演唱会。
      • 用户可使用右下角的保存按钮，将该模式的 General 设置保存到相机的 Customize 内。该操作等同于生成对应模式的二维码，给相机扫描识别保存参数的作用。

  4. 录像

    1. General（录像基础设置）：

      • Mode（拍摄模式）：
        • Normal（普通拍摄）
        • High FPS（高帧率模式，因为是 Binning技术实现，所以画质比普通拍摄的视频略逊色一些）
        • Timelapse（延时摄影）
      • Content Type（内容类型）：
        • 360° Pano
        • 360° 3D
      • Real-time Stitching（实时拼接选项，如果开启后，需要选择实时拼接存储的视频分辨率）
      • Save origins from six lens（存储6个镜头的原片的选项）
      • Single lens resolution（开启存储原片的选项后，可选择每个镜头录制的最大视频分辨率）
      • Flat color mode（后期空间更大的色彩模式，类似于 Log 功能）

    2. Exposure（曝光设置）：

      • Mode：
        • Auto（自动曝光）
        • Manual（手动曝光）
        • Isolated Exposure（各镜头独立曝光）
      • 在自动曝光模式下可调整 EV、白平衡
      • 在手动曝光模式下可调整 ISO、Shutter
      • 在各镜头独立曝光模式下可调整 EV、白平衡
      • 可重置曝光参数为默认设置

    3. Property（画面属性设置）

      • Brightness（亮度）
      • Saturation（饱和度）
      • Sharpness（锐度）
      • Constrast（对比度）
      • 可重置画面属性参数为默认设置

    4. 参数值范围：

      • EV 范围：普通拍摄情况下是 -5~5
      • ISO 范围：100~6400
      • Shutter范围：普通 Shutter 为1/30s~1/8000s
      • WB 档位：Auto（自动白平衡）、2700K、4000K、5000K、6500K、7500K
      • Brightness（亮度）：-255~255
      • Saturation（饱和度）：0~255
      • Sharpness（锐度）：3~6
      • Constrast（对比度）：0~255

    5. 录像功能归纳：

      • 拍摄360°全景/3D普通视频，设置Mode 为 Normal、选择 Content Type，是否开启实时拼接。
      • 如果开启实时拼接，则实时拼接存储的视频最高分辨率不得超过4K。而Content Type 若是此时选择的是360°3D，则原片每路可存储1920x1440分辨率的视频，可使用 Stitcher 后期合成3840x3840 的3D视频。若是此时 Content Type 选择的是360°Pano，则原片每路可存储2560x1440分辨率的视频，可使用 Stitcher 后期合成5120x2560 的普通全景视频。
      • 如果关闭可实时拼接，而Content Type 若是此时选择的是360°3D，则原片每路可存储3200x2400分辨率的视频，可使用 Stitcher 后期合成最高6400x6400 的6K 3D视频。若是此时 Content Type 选择的是360°Pano，则原片每路可存储3840x2160分辨率的视频，可使用 Stitcher 后期合成7680x3840 的8K普通全景视频。
      • 拍摄High FPS高帧率的视频，因为是 Binning技术实现，所以画质比普通拍摄的视频略逊色一些。该模式适合拍摄高速运动或者后期需要调整快放慢放的内容制作场景。Content Type 选择360°3D 的情况下，每路可存储1920x1440@60fps 的原片，可用 Stitcher 合成3840x3840@60fps 的3D 视频。Content Type 选择360°Pano 的情况下，每路可存储1920x1440@60fps 的原片，可用 Stitcher 合成3840x3840@60fps 的3D 视频。
      • 拍摄 Timelapse 延时摄影的作品时，设置好 Content Type，然后设置每张照片之间的间隔时长即可。注意，间隔时长必须设置为2s 以上，而间隔时间至少得比 Shutter 时间长1s，比如设置的间隔时长为2s，那么 Shutter 最慢可设置为1s。
      • Flat color mode（后期空间更大的色彩模式，类似于 Log 功能,用户可以将素材导入后期软件中进行更多的画面细节调整）
      • 曝光参数的设置 中，Auto/Manual 是全局统一的自动/手动曝光。如果选择 Isolated Exposure，则6个镜头将会各自独立的自动曝光。独立曝光适合拍摄明暗差距较大的场景，比如拍摄艳阳高照的窗边与室内，比如舞台绚丽明亮但观众席较暗的演唱会。
      • 用户可使用右下角的保存按钮，将该模式的 General 设置保存到相机的 Customize 内。该操作等同于生成对应模式的二维码，给相机扫描识别保存参数的作用。

  5. 直播推流

    1. General（直播基础设置）：

      • Mode（拍摄模式）：
        • Build-in server（机内服务器推流）
        • Custom RTMP server（自定义 RTMP 推流地址）
        • HDMI output（HDMI 信号输出模式）
        • Aerial（航拍模式，选择该模式，相机将在输出1920x960 HDMI预览画面的同时，保存每路 2560x1440 的原片视频，适用于接 DJI 无人机的图传预览，同时保存录像视频的场景）
      • Content Type（内容类型）：
        • 360° Pano
        • 360° 3D
      • Resolution（直播分辨率）
      • Bitrate（直播推流码率）
      • Live-streaming URL（如果是自定义 RTMP 推流地址的模式，请在此项填写推流地址，由 URL 和 Key 组成）
      • Save origins from six lens（边直播边保存6个镜头原片的开关）
      • Save stitch（边直播边保存直播视频画面）

    2. Exposure（曝光设置）：

      • Mode：
        • Auto（自动曝光）
        • Manual（手动曝光）
        • Isolated Exposure（各镜头独立曝光）
      • 在自动曝光模式下可调整 EV、白平衡
      • 在手动曝光模式下可调整 ISO、Shutter
      • 在各镜头独立曝光模式下可调整 EV、白平衡
      • 可重置曝光参数为默认设置

    3. Property（画面属性设置）

      • Brightness（亮度）
      • Saturation（饱和度）
      • Sharpness（锐度）
      • Constrast（对比度）
      • 可重置画面属性参数为默认设置

    4. 参数值范围：

      • EV 范围：普通拍摄情况下是 -5~5
      • ISO 范围：100~6400
      • Shutter范围：普通 Shutter 为1/30s~1/8000s
      • WB 档位：Auto（自动白平衡）、2700K、4000K、5000K、6500K、7500K
      • Brightness（亮度）：-255~255
      • Saturation（饱和度）：0~255
      • Sharpness（锐度）：3~6
      • Constrast（对比度）：0~255

    5. 直播功能归纳：

      • 直播分为网络推流和 HDMI 信号输出两种形式。
        • 其中网络推流用户可以选择 Build-in server，向相机的 “rtmp://相机IP/live/live” 地址推流，也可以选择 Custom RTMP Server 填写第三方的推流地址，进行外网直播。诸如 Youtube等平台已支持提供 RTMP 全景直播推流地址。
        • HDMI 信号输出模式下，直播流画面以 HDMI 的信号形式对外输出，用户可以使用相机的 HDMI Type-D 接口将画面采集到导播台或者视频采集卡中。而特定的航拍模式下，相机将在输出1920x960 HDMI预览画面的同时，保存每路 2560x1440 的原片视频，适用于接 DJI 无人机的图传预览，同时保存录像视频的场景。
        • 无论是网络推流还算是 HDMI 信号输出，都可以在直播的同时选择保存6路镜头的原片。因为直播推流本质输出的都是实时拼接好的画面，所以直播的最高分辨率不得超过4K。
      • 曝光参数的设置 中，Auto/Manual 是全局统一的自动/手动曝光。如果选择 Isolated Exposure，则6个镜头将会各自独立的自动曝光。独立曝光适合直播明暗差距较大的场景，比如拍摄艳阳高照的窗边与室内，比如舞台绚丽明亮但观众席较暗的演唱会。
      • 用户可使用右下角的保存按钮，将该模式的 General 设置保存到相机的 Customize 内。该操作等同于生成对应模式的二维码，给相机扫描识别保存参数的作用。

  6. 拼接校准

    1. 功能导航页进入第四个功能，Calibration（拼接校准）。该功能的作用是当相机需要直播、或者拍摄实时拼接的内容时，由于具体拍摄场景之间存在差异，会出现拼接错位等情况。例如相机在不同的远景和近景之间的效果会有差别。所以用户预览或者试拍一些作品发现实时拼接的效果不满意，可以用此功能对相机进行校准。
    2. 进入该功能后请按提示将相机放置在一个开阔的地方，并确保相机球半径1m 内不存在遮挡物，包括使用者自己。然后按下 Start 按钮相机将自动会自动基于当前场景进行拼接校准。拼接完成后请查看预览流确认效果。

  7. 存储管理

    1. 使用手机 APP 拍摄完内容后，可在左下角的快捷入口，进入存储管理的相册，查看相机刚拍和历史拍摄的所有素材文件夹。除了 HDR、RAW、Burst、Timelapse 这四种模式的素材没有预览图之外，其余拍摄的作品都有预览图便于用户快速定位。用户可在手机上通过 http 查看单个素材文件，但因为素材文件体积都较大，手机下载和播放的性能都比较低，建议用户还是使用电脑查看管理拍摄的素材。

    2. 存储管理可查看当前相机配备的存储设备的容量使用情况，用户还可以点击左下方的齿轮图标对该存储设备进行速度检测操作

  8. 相机设置（待开发）

    • 用户可通过 APP 修改相机的部分设置，查看相机信息。可设置项与相机保持一致。

四、生成二维码给相机识别

  1. 相机可以在不与手机APP连接的情况下，扫描识别 APP 生成的二维码。

    • 打开 APP 左上角的二维码生成器，可以看到拍照、录像、直播推流、设置WiFi 连接信息四块功能。
    • 其中拍照、录像、直播推流可以设置除了曝光（Exposure）、画面属性（Property）之外的基础设置。请将生成的二维码，对准相机屏幕两侧的镜头的其中一个，等待相机识别。相机扫码识别成功时将会播放识别成功的音效。识别后，即可将参数保存到 Customize 参数档位中，在这之后独立使用相机时，将档位快速切换到 Customize ，即可快速按最近一次的扫码设置来进行拍摄。
    • WiFi 连接信息填写后生成二维码，相机识别后将自动连接。

  2. 请注意，如果需要清除相机已保存的扫码信息（包含 Customize 信息和 WiFi 连接信息），请至相机机身的设置菜单中，用 Reset 功能清除。

  2. 电源与电量状态

    • 相机使用12V 5A 的DC 接口电源适配器
    • 相机使用5000mAh 的可更换电池，不开 WiFi 的情况下续航约75min
    • 关机充电时
      • 当相机未充满电时，电源指示灯红灯长亮
      • 当相机充满电时，电源指示灯绿灯长亮
    • 开机充电时
      • 当相机电量低于10%时，电源指示灯显示为红灯长亮，直至低点保护或完全无电自动关机
      • 当相机电量处于11%~20%时，电源指示灯显示为黄灯亮
      • 当相机电量处于21%~100%时，电源指示灯显示为白灯亮
      • 当相机进行具体工作时，工作灯颜色将会和电源指示灯的颜色保持一致，具体工作状态灯的闪烁状态见相机操作使用一节

          每次拍照时，默认需要等待5s的倒计时，倒计时过程中工作灯将持续闪烁，且伴有音效提示，请在这5s内远离相机，尽量避免太过靠近相机，影响拍照效果。拍照成功时将有完成音效的提示，

  6. 拼接校准

    • 相机的机内拼接效果取决于具体拍摄场景。比如相机在不同的远景和近景之间的效果会有差别。所以用户预览或者试拍一些作品发现实时拼接的效果（直播、录像实时拼接时）不满意，可以用相机的这个功能校准。进入该功能后请按提示在5秒内远离相机1米远，以便于倒计时结束后的拼接校准可以获得最好的效果。

  7. 相机设置

    1. 设置相机的 WiFi 模式

      • Pro 相机提供了两种 WiFi 连接的方式：自身 AP 热点和连接外部无线 WiFi

      • 设置为 AP，然后开启首页的 WiFi 按钮，则屏幕顶部会出现WiFi 开启的标志并且显示一个 IP 地址。此时用户使用客户端设备搜寻附近的相机 AP，默认密码88888888（8个8），可以进行连接。

      • 设置为 WiFi，即认为连接外部 WiFi 的模式，选中首页的 wifi 按钮，按下设置键，将进入等待用户扫描相机 WiFi 信息（SSID 与密码）的界面。用户可以使用手机 APP 的二维码生成功能，设置好要连接的外部WiFi 的 SSID 与密码，生成二维码给相机扫描。扫描成功后相机将自动连接至该外部网络。详情请见手机 APP 二维码扫描部分。

    2. 设置相机Ethernet 网线连接

      • DHCP：用于将 Pro 连入某个局域网络，动态获取网络内的 IP 地址
      • Direct：用于将 Pro和 PC/笔记本电脑 直接连接
        具体用法：
        1. 使用网线将 pro 与电脑的网口直接相连。如果需要连接的电脑没有网线接口，可以使用 pro 配备的网线与网卡接入电脑的 USB 接口。
        2. 将 pro 的相机设置中，Ethernet 选项切换为“Direct”模式。此时，相机顶部的 IP 地址将会固定显示为“192，168.1.188”
        3. 将另一端连接的电脑，有线网络的IP 地址获取方式设置为手动，将电脑与 Pro 设置在同一个网段。
          • Mac OS：
            • 打开网络偏好设置
            • 选择USB 网卡网络，将配置 IPV4选项设置为“手动”，将 IP 地址设置为“192.168.1.100”，子网掩码设置为“255.255.255.0”
            • 应用保存
          • Windows：
            • 打开本地网络，右键菜单选择属性，在弹出界面中选择 IPV4地址设置。
            • 如图，将 IP 地址设置为“192.168.1.100”，子网掩码设置为“255.255.255.0”
            • 点击保存
        4. 完成以上步骤之后，可以照常的使用电脑端的 Pro 控制软件去操控相机。
    3. FlatColor：视频的 FlatColor 模式，开启此项后将会得到更适合后期处理的视频画面
    4. Frequency：用于设置视频画面的刷新频率。50Hz、60Hz，分别适合不同电力频率的地区使用

    5. Gyro Calibration：陀螺仪校正功能。Pro 相机支持陀螺仪数据记录用于拍摄、后期处理时的防抖与水平校正。如果用户预览、试拍时发现相机画面的陀螺仪水平防抖效果不好，可通过此功能来校正

    6. Storage: 存储管理，用户可使用此功能查看当前存储卡或者USB 存储设备的容量使用情况。还可以选择插入的设备进行格式化、速度检测等操作

    7. LED: LED 开关，用户可开启或关闭相机周身指示灯
    8. Speaker: Speaker 开关，用户可开启或关闭相机的扬声器提示音
    9. Bottom Logo：设置画面底部是否 Logo 遮挡
    10. Info: 查看相机硬件固件、系统等程序版本信息
    11. Reset: 将相机记录的Customize参数重置为出厂默认

  8. 如何设置与修改底部Logo？

    • 请在相机设置中将 Bottom Logo 选项开启。
    • 将命名为”_logo.png”的 Logo 图拷贝至相机当前使用的 SD 卡或移动硬盘。
    • 打开相机观看预览流或者拍摄一个实时拼接的内容，相机会自动将”_logo.png”转化成底部的圆形遮挡水印。
    • “_logo.png”要求宽高比最大不超过50%。在宽度固定的情况下，增大高度将增大底部 logo 的半径。

  10. 二维码扫描功能

    • 相机可以在不与手机APP连接的情况下，扫描识别 APP 生成的二维码，对准相机屏幕两侧的镜头的其中一个，等待相机识别后，即可将参数保存到 Customize 参数档位中，以便进行更多操作。相机扫码识别成功时将会播放识别成功的音效。

    • 在拍照、录像、直播推流的准备模式下，按下设置键，将会进入等待扫描二维码状态。请使用 APP 的二维码生成器，设置对应模式的参数，并生成二维码给相机识别。

    • 在首页导航栏中选中 WiFi 图标，按下设置键即可进入设置等待扫描外部 WiFi 信息的二维码状态。请使用 APP 的二维码生成器，设置外部 WiFi的 SSID 和密码，并生成二维码给相机识别。相机在识别成功后将自动连接填写的外部 WiFi 信息
